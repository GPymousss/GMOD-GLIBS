## 🇫🇷 Français

Si vous obtenez des erreurs liées à un `require(...)`, c'est normal : vous devez installer le module `.dll` de MySQLOO dans votre dossier `/garrysmod/lua/bin/`.

⚠️ **Attention** : il existe 4 fichiers `.dll` différents selon votre système d'exploitation et l'architecture de Garry's Mod :

| Système | Lien |
|---|---|
| Windows 32 bits | https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_win32.dll |
| Windows 64 bits | https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_win64.dll |
| Linux 32 bits | https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_linux.dll |
| Linux 64 bits | https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_linux64.dll |

Si vous rencontrez toujours des problèmes avec le `require()`, n'hésitez pas à m'envoyer un message privé sur Discord : **gpymousss**

> Pour tout problème non lié à ma librairie, je ne pourrai malheureusement pas vous aider.
