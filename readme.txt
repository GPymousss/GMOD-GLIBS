Si vous avez des erreurs à cause d’un require(...), c’est normal : il faut installer le module .dll de MySQLOO dans votre dossier /lua/bin/.
⚠️ Attention : il existe 4 fichiers .dll différents selon votre système d’exploitation (Windows/Linux) et l’architecture de votre machine ou de Garry’s Mod (x32/x64) :

Windows 32 bits : https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_win32.dll
Windows 64 bits : https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_win64.dll
Linux 32 bits : https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_linux.dll
Linux 64 bits : https://github.com/FredyH/MySQLOO/releases/latest/download/gmsv_mysqloo_linux64.dll
